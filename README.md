### IE10 Input height
You don't see the value in your inputs? Then check your padding and height.

#### Example:

input {
    height:         32px;
    padding-top:    16px;
    padding-bottom: 16px;
}

#### Solution:
input {
    height: auto;
    /* OR lesser padding */
    padding-top:    15px;
}
